# SQL-запросы проекта

**Дата создания:** 2025-12-19 11:52 (UTC+3, Брест)  
**Версия:** 1.0  
**Описание:** Каталог SQL-запросов для REST приложения Bitrix24

---

## Структура

### original-queries/
Исходные SQL-запросы (как есть в коде).

**Формат файла:** `[TASK-ID]_[METHOD].sql`

**Метаданные в шапке:**
```sql
-- Источник: app/Services/LeadSyncService.php::sync()
-- Дата: 2025-12-19 11:52 (UTC+3, Брест)
-- Тип: SELECT
-- Описание: Получение лидов для синхронизации
```

### optimized/
Оптимизированные SQL-запросы.

**Формат файла:** `[TASK-ID]_[METHOD]_v2.sql`

**Метаданные в шапке:**
```sql
-- Оптимизировано: 2025-12-19 11:52 (UTC+3, Брест)
-- Причина: Добавлен индекс по `created_at`
-- Ссылка на задачу: TASK-005
```

### history/
История изменений SQL-запросов.

**Формат:** Markdown файлы с описанием изменений.

---

## Типы запросов

- **SELECT** — чтение данных
- **INSERT** — вставка данных
- **UPDATE** — обновление данных
- **DELETE** — удаление данных
- **JOIN** — связи между таблицами
- **AGGREGATE** — агрегация данных

---

## Примечание

REST приложение Bitrix24 работает с данными через **Bitrix24 REST API**, а не напрямую с базой данных. SQL-запросы могут потребоваться только для:
- Локального кеширования данных
- Аналитики и отчётов
- Интеграции с внешними системами
