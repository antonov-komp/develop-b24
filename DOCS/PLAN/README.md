## Структура

### roadmap.md
Дорожная карта проекта с основными этапами развития.

### milestones.md
Конкретные этапы (милстоуны) проекта с датами и критериями выполнения.

---

## Формат планов

### Дорожная карта
```markdown
## Этап 1: Базовая функциональность
- Описание этапа
- Сроки: YYYY-MM-DD
- Задачи: TASK-001, TASK-002

## Этап 2: Расширенная функциональность
- Описание этапа
- Сроки: YYYY-MM-DD
- Задачи: TASK-003, TASK-004
```

### Милстоуны
```markdown
## Милстоун 1: Название
**Дата:** YYYY-MM-DD  
**Статус:** Запланирован / В работе / Завершён

### Критерии выполнения
- [ ] Критерий 1
- [ ] Критерий 2

### Задачи
- TASK-001
- TASK-002
```
