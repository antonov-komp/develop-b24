## Обзор

**Calendar-модуль** Bitrix24 предоставляет методы для работы с календарём, событиями, участниками и напоминаниями.

**Документация:**
- **REST API:** https://context7.com/bitrix24/rest/calendar/

---

## Основные методы

### События
- `calendar.event.get` — получение события
- `calendar.event.add` — создание события
- `calendar.event.update` — обновление события
- `calendar.event.delete` — удаление события
