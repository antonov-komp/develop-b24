## Обзор

**Files-модуль** Bitrix24 предоставляет методы для работы с файлами, облачным хранилищем (Disk), версиями файлов и поиском.

**Документация:**
- **REST API:** https://context7.com/bitrix24/rest/disk/

---

## Основные методы

### Диск (Disk)
- `disk.folder.getchildren` — получение содержимого папки
- `disk.file.uploadversion` — загрузка новой версии файла
- `disk.file.get` — получение информации о файле
- `disk.file.delete` — удаление файла
