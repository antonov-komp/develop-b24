# Tasks — методы работы с задачами в Bitrix24

**Дата создания:** 2025-12-20 19:09 (UTC+3, Брест)  
**Версия:** 1.0  
**Описание:** Документация по методам работы с задачами и проектами

---

## Обзор

**Tasks-модуль** Bitrix24 предоставляет методы для работы с задачами, проектами, комментариями и исполнителями.

**Документация:**
- **REST API:** https://context7.com/bitrix24/rest/tasks/

---

## Основные методы

### Задачи
- `tasks.task.list` — получение списка задач
- `tasks.task.get` — получение задачи по ID
- `tasks.task.add` — создание задачи
- `tasks.task.update` — обновление задачи
- `tasks.task.delete` — удаление задачи
- `tasks.task.fields` — получение полей задачи

### Комментарии
- `tasks.commentitem.getlist` — получение комментариев задачи
- `tasks.commentitem.add` — добавление комментария

### Проекты
- `tasks.task.get` — получение информации о проекте
- `tasks.task.list` — получение списка проектов

---

## Примеры использования

### Создание задачи

```php
$result = $client->call('tasks.task.add', [
    'fields' => [
        'TITLE' => 'Новая задача',
        'DESCRIPTION' => 'Описание задачи',
        'RESPONSIBLE_ID' => 1,
        'CREATED_BY' => 1,
        'DEADLINE' => '2025-12-31 23:59:59'
    ]
]);
```
