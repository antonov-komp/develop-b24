# MCP (Mobile Client Protocol) — протокол для мобильных приложений

**Дата создания:** 2025-12-20 19:09 (UTC+3, Брест)  
**Версия:** 1.0  
**Описание:** Документация по MCP для работы с мобильными приложениями Bitrix24

---

## Обзор

**MCP (Mobile Client Protocol)** — протокол для разработки мобильных приложений Bitrix24. Оптимизирован для работы с мобильными устройствами, поддерживает офлайн-режим и инкрементальную синхронизацию.

**Документация:**
- **Официальная:** https://apidocs.bitrix24.ru/sdk/mcp.html

---

## Когда использовать

✅ **Используйте MCP, если:**
- Разрабатываете нативное мобильное приложение
- Нужна офлайн-синхронизация данных
- Требуется оптимизация для мобильных устройств

❌ **Не используйте MCP, если:**
- Разрабатываете веб-приложение (используйте REST API)
- Нужна простая интеграция (используйте REST API)

---

## Основные принципы

### 1. Инкрементальная синхронизация

MCP поддерживает синхронизацию только изменённых данных, что снижает объём передаваемых данных.

### 2. Офлайн-режим

Поддержка работы без подключения к интернету с последующей синхронизацией.

### 3. Сжатие данных

Автоматическое сжатие данных для снижения трафика.
