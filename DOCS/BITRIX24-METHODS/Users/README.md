---

## Обзор

**Users-модуль** Bitrix24 предоставляет методы для работы с пользователями, группами, правами доступа и профилями.

**Документация:**
- **REST API:** https://context7.com/bitrix24/rest/user/

---

## Основные методы

### Пользователи
- `user.get` — получение пользователя по ID
- `user.current` — получение текущего пользователя
- `user.search` — поиск пользователей
- `user.admin` — проверка прав администратора

### Группы
- `sonet_group.get` — получение группы
- `sonet_group.user.groups` — получение групп пользователя

---

## Примеры использования

### Получение текущего пользователя

```php
$result = $client->call('user.current');
$user = $result['result'] ?? null;
```
