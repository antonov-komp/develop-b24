**Дата создания:** 2025-12-20 19:09 (UTC+3, Брест)

## Обзор

**IM-модуль** Bitrix24 предоставляет методы для работы с мессенджером, чатами, уведомлениями и историей сообщений.

**Документация:**
- **REST API:** https://context7.com/bitrix24/rest/im/

---

## Основные методы

### Сообщения
- `im.message.add` — отправка сообщения
- `im.chat.get` — получение информации о чате
