# Timeman — методы работы с учётом времени в Bitrix24

**Дата создания:** 2025-12-20 19:09 (UTC+3, Брест)

---

## Обзор

**Timeman-модуль** Bitrix24 предоставляет методы для работы с учётом рабочего времени, отметками начала/окончания работы, перерывами и отчётами.

**Документация:**
- **REST API:** https://context7.com/bitrix24/rest/timeman/

---

## Основные методы

### Учёт времени
- `timeman.status` — получение статуса учёта времени
- `timeman.open` — начало рабочего дня
- `timeman.close` — окончание рабочего дня
